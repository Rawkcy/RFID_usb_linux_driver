Read from the interrupt endpoint.

Parameters:
[in]hidifWhich interface to send to
[in]epWhich endpoint to read
[out]bytesBuffer to store results
[in]sizeHow many bytes to read
[in]timeoutHow long to wait, in milliseconds

Upon successful completion, the bytes array will contain an input report from the device. It is up to the programmer to determine which input endpoint to read, and to ensure that the endpoint is bitwise-ORed with 0x80 (USB_ENDPOINT_IN).

hid_return hid_set_idle ( HIDInterface *const  hidif,
unsigned  duration,
unsigned  report_id 
)

Execute a Set_Idle request on an Interrupt In pipe.

This is used to tell a device not to send reports unless something has changed (duration = 0), or unless a minimum time interval has passed.

Parameters:
[in]hidifWhich interface to send to
[in]duration0 for indefinite, otherwise in increments of 4 ms (to 1020 ms)
[in]report_id0 for all reports, otherwise a Report ID
